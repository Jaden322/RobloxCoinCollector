# RobloxCoinCollector by Jaden
Beginner Roblox game project: players collect coins when they touch them. Written in Lua U.

## Project Description
This is a beginner Roblox project of mine where players collect coins when they touch them. Written in Lua.

## How It Works
- Players touch a coin object
- The coin gives 10 points to the player's total
- The coin is destroyed after collection to prevent multiple collections

## What I Learned
- How to use functions and variables in Lua
- How to handle events (Touched)
- Debugging scripts in Roblox Studio
